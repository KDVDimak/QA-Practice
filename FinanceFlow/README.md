# Finance Flow — QA Testing

This folder contains QA documentation for the **Finance Flow** web application (online wallet).

---

## Application Overview

Finance Flow allows users to:
- Register and log in
- Add income and expense transactions
- View and manage balance

---

## Testing Scope

- User authentication
- Transactions (add / delete)
- Balance calculation
- Input validation
- Basic UI behavior

---

## Test Artifacts

- **test-cases.md** — functional test cases  
- **bug-reports.md** — documented issues  
- **checklist.md** — quick testing checklist  

---

- Frontend: Web
- Backend: Node.js
